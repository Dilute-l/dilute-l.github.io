题解 CF1051E Vasya and Big Integers

哈希 + 二分 + DP

首先看到题面，很容易想到一个DPDPDP，令f[i]f[i]f[i]为划分到iii为止的方案数。

然后朴素的暴力转移是O(n2)O(n^2)O(n2)的，非常显然一个状态iii能够转移到的jjj是一段连续的，进而想到使用前缀和优化。

令lll的长度为slslsl，rrr的长度为srsrsr，那么长度为lenlenlen（sl<len<srsl < len < srsl<len<sr）的一段数字sss必定满足l<s<rl < s < rl<s<r。然后那么我们只需要考虑当前状态iii的时候[i,i+sl)[i, i + sl)[i,i+sl)和[i,i+sr)[i,i + sr)[i,i+sr)这两段区间分别和l,rl,rl,r的大小关系。

如果我们我们要比较两个字符串aaa和bbb的大小关系，我们完全先特判掉两个完全一样的情况，然后再求出它们的lcplcplcp，然后那么aaa和bbb的大小关系就是a[lcp+1]a[lcp + 1]a[lcp+1]和b[lcp+1]b[lcp + 1]b[lcp+1]的大小关系了。

lcplcplcp可以哈希 + 二分解决，接下来的事情就是DPDPDP的了
